# Legacy HTML Deprecated

The previous standalone H5 entry (`index.html` + `js/*.js` + `css/style.css`) is deprecated.

All new feature development should continue in the Uni-app structure:

- `App.vue`
- `main.js`
- `pages.json`
- `pages/index/index.vue`
- `src/config/*`
- `src/services/*`

Legacy files are temporarily retained only for reference during migration.
